# 屏幕伴侣插件更新日志

## 2.5.3 (2026-03-13)

### 新增
- 新增“共同经历”长期记忆，记录用户明确提到的与 Bot 一起看过的影视、玩过的游戏、做过的测试和识屏互动
- WebUI 新增表格化统计看板
- 统计看板新增时间范围筛选，支持今天、近 7 天、近 30 天、全部时间和自定义起止日期

### 优化
- 优化开始/结束任务的 LLM 回复，使其更稳定地应用人格语气，减少机械式系统播报
- 开始/结束任务回复现在会优先使用当前会话的人格上下文
- README 与 WebUI 版本信息同步更新到 `2.5.3`

## 2.5.2 (2026-03-11)

### 新增
- 实现了错误纠正和持续性记忆功能
- 实现了自我形象识别和记忆
- 实现了 `webui` 命令来返回端口信息

### 修复
- 修复了 "TCPSite.__init__() got an unexpected keyword argument 'sock'" 错误
- 修复了 "charset must not be in content_type argument" 错误
- 修复了 "json_response() got an unexpected keyword argument 'charset'" 错误
- 修复了 WebUI 中的中文乱码问题

### 优化
- 优化了 `_build_vision_prompt` 方法，按重要性排序组织提示词，以降低 LLM 反应时间
- 更新了默认端口从 8898 到 6314
- 减少了自动切换端口的次数从 10 到 3

## 2.5.1 (2026-03-11)

### 修复
- 修复了 `window_change_cooldown` 属性缺失导致的任务异常
- 修复了窗口变化检测相关属性的初始化问题

### 优化
- 删除了突兀的语气词和情感词汇，提升对话流畅度
- 优化了LLM提示词顺序，提高回复速度和对话连贯性
- 减少了提示词长度，提高处理效率
- 改进了提示词结构，增强上下文理解

- WebUI 端口重载时的释放与重试逻辑
